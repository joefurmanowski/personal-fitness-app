  <br />

<h4 align="center">A C++ application focused on personal fitness that helps users keep track of their health and activity based on user input.</h4>

## Key Features

- Input fitness activity details and save it to a file
  - Current weight
  - Type of exercise
  - Duration of exercise
- Display latest fitness activity input
- Display history of inputs in descending order, read from a file
- Display summary report

## Key Implementations

- `iostream`
  - Standard input/output streams
- `fstream`
  - File streams (read/write)
- `iomanip`
  - Stream manipulators
- `vector`
  - Support for dynamic size arrays
